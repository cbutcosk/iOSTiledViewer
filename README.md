## Installation

iOSTiledViewer is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following lines to your Podfile:

```ruby
source 'https://github.com/Fiser33/iOSTiledViewerSpec.git'
pod "iOSTiledViewer"
```
